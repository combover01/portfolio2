Binauralize Your Beats
Group project members: Mir Jeffres, Tucker Alexander, Duncan Hanson, Kai White
Code repository: https://github.com/combover01/binauralize-your-beats

This project was a website that would turn any uploaded song into a binaural beats audio by pitch-shifting the bass frequencies to get to the desired binaural beat frequency. It was built with Flask and Jinja2. I will have it running on this site soon, but in the meantime if you want to try it out, just clone the repo above and build the Flask site using the Flask tutorial documentation.
